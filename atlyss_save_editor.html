Save Editor





Selected Files:

    Edit File Content:

    1. You can change the language by clicking Select Language.
    2. (Optional) Click Configuration File to load a settings file for the save editor.
    3. Click Load Folder to select the save folder for ATLYSS.
      • By default, the save files are located at:
        [Your Steam installation path]\Steam\steamapps\common\ATLYSS\ATLYSS_Data\profileCollections
      • This script runs locally on your computer and does not upload any data to servers or external locations.
    4. Use Select a file to choose the file you want to edit.
    5. Once selected, the file will be loaded.
    6. Make the modifications you need.
    7. Click Download Modified File to save the edited file.
      • The downloaded file needs to have the .json file extension removed.
    8. Back up your save folder and store it in a safe place.
    9. Replace the original save file in the save folder with the modified file.
    10. Launch the game and enjoy your edits!
